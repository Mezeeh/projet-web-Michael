Nightmare!

Auteur : Eowaven

Description

Nightmare! ajoute à Witcher 3 une nouvelle difficulté, comme son nom peu vous lavoir fait remarqué, extrêmement dur ou si le joueur meurt il doit recommenser à sa dernière sauvegarde. Il est seulement possible de sauvegarder dans un lit durant la nuit et il est impossible d'utiliser la fonction pour passer le temps.

Commment installer le mod

Télécharger le fichier

Extraire le fichier avec 7-Zip ou winRAR

Copier le fichier extrait dans le dossier mods de votre jeu ex: S:\Program Files (x86)\GalaxyClient\Games\The Witcher 3 Wild Hunt\Mods

Lancer le jeu
