Andúril la « Flamme de l'Ouest »

Auteur : Cerann

Description

Andúril la « Flamme de l'Ouest » est un mod qui ajoute la lame précédement appelée Narsil qui a été brisé puis reforgé par Elrond pour Aragorn II Elessar dans le film Le Seigneur des anneaux : Le Retour du roi à Witcher 3.

Commment installer le mod

Télécharger le fichier

Extraire le fichier avec 7-Zip ou winRAR

Copier le fichier extrait dans le dossier mods de votre jeu ex: S:\Program Files (x86)\GalaxyClient\Games\The Witcher 3 Wild Hunt\Mods

Lancer le jeu
